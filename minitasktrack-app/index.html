Задачи, specs и критерии приемки

Рабочая доска для маленькой SaaS-команды: задача превращается в короткий product spec, чек-лист разработки и acceptance criteria без тяжелой CRM.

Выберите задачу

Клик по карточке покажет spec, чек-лист и acceptance criteria. Все изменения сохраняются в localStorage.

Product spec

  • Сформулируйте пользовательскую проблему.
  • Опишите ожидаемое поведение продукта.
  • Проверьте критерии приемки перед релизом.

Dev checklist

  • UI состояние пустого списка.
  • Состояние ошибки и загрузки.
  • Мобильная проверка.

Acceptance criteria

  • Пользователь понимает следующий шаг.
  • Ключевой сценарий завершается без ручных обходов.
  • Регрессия не ломает релизный поток.
